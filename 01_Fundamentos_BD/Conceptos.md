# Conceptos Básicos

## ¿Qué es una Base de Datos?

Una base de datos es un conjunto organizado de información que permite almacenar, consultar, modificar y eliminar datos de manera eficiente.

Su objetivo principal es mantener la información organizada, segura y disponible para los usuarios.

---

## ¿Qué es un Sistema Gestor de Bases de Datos (DBMS)?

Es el software encargado de administrar una base de datos.

Permite:

- Crear bases de datos.
- Gestionar usuarios.
- Ejecutar consultas.
- Controlar la seguridad.
- Garantizar la integridad de la información.

### Ejemplos

- MySQL
- PostgreSQL
- SQL Server
- Oracle Database
- MongoDB (NoSQL)

---

## ¿Qué es una Tabla?

Es una estructura donde se almacenan los datos.

Cada tabla representa una entidad del negocio.

Ejemplo:

Tabla Clientes

| id | nombre | correo |
|----|---------|---------|
|1|Ana|ana@email.com|

---

## ¿Qué es un Registro?

Es cada fila de una tabla.

También se conoce como:

- Tupla
- Row

---

## ¿Qué es un Campo?

Es cada columna de una tabla.

También se conoce como:

- Columna
- Atributo

---

## ¿Qué es una Consulta?

Es una instrucción que permite obtener información almacenada en la base de datos.

Generalmente se realiza mediante SQL.

---

## ¿Qué es la Integridad de los Datos?

Es el conjunto de reglas que garantizan que la información almacenada sea correcta, consistente y válida.

---

## Buenas prácticas

✅ Nombrar las tablas en singular.

✅ Utilizar nombres descriptivos.

✅ Evitar información duplicada.

✅ Definir correctamente las claves primarias.
